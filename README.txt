BEM VINDO AO JOGO DA ADVINHAÇÃO v1.0

COMO EXECUTAR:

AO BAIXAR A PASTA PYTHON-JOGO DEVERÁ ABRIR A PASTA JOGO_DA_ADVINHAÇÃO E CLICAR NO EXECUTÁVEL DO JOGO

-> Jogo_da_Advinhação.exe

CASO QUEIRA, TAMBÉM PODERÁ CRIAR UM ATALHO PARA A ÁREA DE TRABALHO


REGRAS PDO JOGO:

- AO INICIAR O JOGO, O USUÁRIO DEVERÁ ESCOLHAR A QUANTIDADE DE RODADAS QUE DESEJA JOGAR

- DETERMINADO O NÚMERO DE RODADAS, O JOGO VAI INICIAR A 1° RODADA E PEDIRÁ O CHUTE.

- A CADA CHUTE ERRADO O JOGO IRÁ INFORMAR SE O CHUTE É MAIOR OU MENOR DO QUE O NÚMERO SECRETO

- AO FINAL DA RODADA, CASO NÃO TENHO ACERTADO, O JOGO INFORMARÁ O NÚMERO E PERGUNTARÁ SE DESEJA JOGAR NOVAMENTE
